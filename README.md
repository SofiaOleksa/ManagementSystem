# ManagementSystem

Система для управління **Users, Items, Actions** з MVC-адмін-панеллю (ASP.NET Core) та клієнтом **.NET MAUI**.

## Вимоги
- Visual Studio 2022 Community
- .NET 8 SDK
- Workloads: **ASP.NET and web development**, **.NET MAUI**
- Android Emulator **API 33 (Tiramisu)**
